Para ejecutar el Script1, basta poner el comando 'Python3 Script1.py' desde la linea de comandos de la terminal.

Al ejecutar se solicitará el nombre de ambos jugadores y luego un número para la cantidad de sets a jugar. Posteriormente, se le solicita al usuario ingresar el jugador que anota punto, para ello se debe ingresar el número de jugador (ya sea 1 o 2). 

Si se ingresa algo equivocado, el programa no permite seguir adelante hasta que se ingrese un dato correcto.
